║           VexaMiner Firmware Flash Instructions                  ║
║           Board: ESP32-2432S028R (Standard Version)              ║
╚══════════════════════════════════════════════════════════════════╝

═══════════════════════════════════════════════════════════════════
STEP 1: Download ESP Flash Download Tool
═══════════════════════════════════════════════════════════════════

Download from:
https://www.espressif.com/en/support/download/other-tools

Or use the one included in: bin/flash_download_tool_3.9.3/

═══════════════════════════════════════════════════════════════════
STEP 2: Configure Flash Download Tool
═══════════════════════════════════════════════════════════════════

1. Run: flash_download_tool_3.9.3.exe
2. Select: ChipType = ESP32
3. Select: WorkMode = develop

═══════════════════════════════════════════════════════════════════
STEP 3: Add Firmware Files
═══════════════════════════════════════════════════════════════════

Check (☑) and add these files with their addresses:

   ☑  bootloader.bin      @  0x1000
   ☑  partitions.bin      @  0x8000
   ☑  boot_app0.bin       @  0xe000
   ☑  firmware.bin        @  0x10000

═══════════════════════════════════════════════════════════════════
STEP 4: Configure Settings
═══════════════════════════════════════════════════════════════════

SPI SPEED:      40MHz
SPI MODE:       DIO
FLASH SIZE:     32Mbit (4MB)
COM PORT:       (Select your COM port - usually COM3, COM4, etc.)
BAUD:           921600 (or 115200 if 921600 fails)

═══════════════════════════════════════════════════════════════════
STEP 5: Enter Boot Mode (If Required)
═══════════════════════════════════════════════════════════════════

Some boards may need manual boot mode:

   1. Press and HOLD the BOOT button on the board
   2. While holding BOOT, press the RESET button
   3. Release RESET, then release BOOT
   4. Click START in Flash Download Tool

═══════════════════════════════════════════════════════════════════
STEP 6: Start Flashing
═══════════════════════════════════════════════════════════════════

1. Click "START" button
2. Wait for "FINISH" message
3. Board will restart automatically

═══════════════════════════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

❌ Connection Failed:
   - Try lower baud rate (115200)
   - Enter boot mode manually (see Step 5)
   - Check USB cable and COM port

❌ Flash Failed:
   - Erase flash first (use "ERASE" button)
   - Try different USB port
   - Check board power

❌ Screen Dark After Flash:
   - Press RESET button on board
   - Power cycle the board
   - Check display cable connection

═══════════════════════════════════════════════════════════════════
VERIFICATION
═══════════════════════════════════════════════════════════════════

After successful flash, the board should show:

1. "VexaMiner" boot splash (white "Vexa" + orange "Miner")
2. WiFi setup screen (if first boot)
3. Mining screen with stats (after configuration)
